UART / APB Peripheral Complete

A configurable UART transmitter/receiver wrapped in an AMBA APB register interface — the first building block toward a full SoC.

  • Register map (TXDATA / RXDATA / STATUS / BAUDDIV) implemented as a clean APB slave
  • 16× oversampled baud generator, FSM-based TX and RX shift registers with 2-flop input synchronization
  • Self-checking SystemVerilog testbenches — 44/44 TX checks and 4/4 RX checks passed across 4 data patterns (0xA5, 0x00, 0xFF, 0x55)
  • Synthesized, implemented, and programmed onto a real Nexys A7 (Artix-7 XC7A100T) via a board-level wrapper driven by switches and LEDs
